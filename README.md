# About this repository

This is a Python re-creation of a small entry to the 2005 Interactive Fiction [Competition](https://ifcomp.org/comp/2005). That [game](https://ifdb.org/viewgame?id=k90xf7mt4wgar7ma), by the [creator](https://strivenword.com/) of this repository, was made as a consequence of learning to code by following the examples in the [Inform Beginner's Guide](https://www.ifwiki.org/The_Inform_Beginner's_Guide) by Roger Firth and Sonja Kesserich. The IBG is a truly great programming book and has provided abundant benefit to the present author, and likely to the community well beyond the confines of the IF forum scene.

This Python reconstruction was motivated by the goal of learning Python, properly and thoroughly, of building muscle memory in writing and thinking through code in a Python native way.
